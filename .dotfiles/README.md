## Install

```bash
git clone --bare git@github.com:sammyfischer/dotfiles.git "$HOME/.dotfiles"
"$HOME/scripts/dotfiles-setup.sh"
```

Use `dot add <file>` to begin tracking a file

There are some useful aliases in bashrc
- `dotadd` to add all changes from tracked files
- `dotpatch` to add changes interactively
- `dotfetch` to fetch everything
